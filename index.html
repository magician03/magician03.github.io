BlogSpace

MultiUser Blogging Platform

A simple blogging platform for everyone to post and share their blogs.
I built it using the 'Google WebApp2' python library for backend, BootStrap, CSS, HTML, JavaScript for FrontEnd and Google DataStore for Database management. User Accounts and Security is taken care of. Blogpost only take text without any images and stored as String in the DataStore. It is now in production on Google Cloud Platoform.

Time Period: Summer 2016

Gather

Cross Platform Event-Organisation services

Gather is an cross-platform Event-Organisation service where anyone can host their events, attend any events in their desired localities, and view all the events happening around them.
The rudimentary version of this project is done HackInTheNorth hackathon organised by IIIT-Allahabad. Later enhanced and extended it to other platforms like Android and Messenger bots in another hackathon IIITA-Hacks organised by same college. It uses PHP as backend, built using MVC framework, MySQL for storing data and BootStrap for the front-end. It is also available in Google PlayStore. Sample service of the products(Web, Android) can be shown upon request as I host it for temporarily on sandboxed servers.

Time Period: 3 days (April 1-2, 2016; September 10, 2016)

IIIT-A Hacks Website

Official website of Intra-College Hackathon

It is the official website of Intra-College hackathon. This is a static site with BootStrap, HTML, CSS, JS and various other JQuery plugins. The attractive color-changing background is done using CSS-KeyFrame Animation. The site also has Facebook like button embedded in it.

Time Period: September first week, 2016

Online Judge

Google CodeJam style

Online Judge on the lines of Google CodeJam.
This uses PHP for the backend, BootStrap, HTML, CSS, JS for front-end, MySQL for databases. Built on MVC framework. The output files are compared using their checksums. I built the entire website as a part of Tri-Hacker contest in our college.

Time Period: April 2016

Youtube Instant

Inspired from Google Instant Search

Browse and play YouTube videos instalty like Google Instant Search. This uses React for component renderring in the views, WebPack-dev-server and Node for controller part. This doesnt use any Database, instead uses Youtube Search API for browing the YouTube. It consists of 3 major components, search bar, video-list and currently-playing video. Built this following a Udemy course on React.

Time Period: Summer 2016

Personal Website

Developer Portfolio cum Blog

My personal website cum Blog. The website is a simple static. Attractive background gradient animation is generated here.. WebPage also uses various JQuery plugins. My yet-to-start blog is hyper-linked in the floating nav-bar. The blog uses Harp as Static-site generator which in-turn is NodeJS based. All the articles are written in MarkDown format(Jade can also be used!). The site is hosted for free on Github Pages under user site.

Time Period: August-September 2016

Finance & Trading

This a coursework of Harvard CS50 PSet7. I should abide the Honor Code and couldnt reaval much of the details regarding this project. PeerToPeer Chat

Super secure P2P chat service.

Super secure chat service using Peer to Peer technology. Built with NodeJS and NPM modules written by Mafintosh. This does not use any database and runs on command line interface. All the usernames are temporary and will be vanished after the session. It also support basic level group chat and indivial private chat. It does not contain logs of any conversation and is super secure for confidential messages.

Time Period: August 2016
